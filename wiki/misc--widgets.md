# Widgets


## Reload Widget

 **+** 

Press Command + R at the same time and release. The Widget will spiral and reload it's self.

## FAQ

### Systran Translation Widget

The Widget configuration files have been corrupted.

Delete the following files:

```bash
/Users/<juser>/Library/Preferences/widget-com.apple.widget.translation.plist
/Users/<juser>/Preferences/widget-com.apple.widget.translation.plist

```


Once the configuration files have been deleted remove and re-add the Widget to the dashboard.
